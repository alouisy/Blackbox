DOCUMENTATION DU JEU BLACKBOX



Lorsque vous démarrez ce jeu vous arrivez sur le menu, cinq choix vous s'on proposé:



PARTIE SOLO

Avant de commencer à jouer, vous devez choisir un niveau de difficulté, celui ci va de Facile, à Dieux Grecs. A vous de choisir quel niveau vous convient le mieux.
Dans ce mode de jeu l'ordinateur place des atomes aléatoirement et vous devez les trouvez en cliquant sur les cases rouges. Lorsque cous cliquez sur l'une des cases rouge un laser part de cette case en ligne droite, il peut être soit dévié, soit absordé, soit réfléchis, sinon il continue sa course en ligne droite. Si vous avez activé l'option "trajet des lasers" vous pourrez voir le parcours des lasers, cela vous aidera à mieux deviner la position de chaques atomes. Le nombre d'atome à trouver est ecrit au dessus du plateau de jeu. Si le rayon est dévié, absordé ou réfléchit, un message apparaitra en dessous du plateau de jeu. Vous avez aussi accès aux options grace à l'icone en haut à gauche, celles ci vous permettrons par exemple de changer le volume de la musique, de changer la musique, ou encore de changer la couleur de fond. En cliquant sur le nom par défaut "Joueur 1", vous pourrez entrez le pseudonyme que vous souhaitez. Lorsque vous avez terminé et pensez avoir trouvé tous les atomes vous pouvez cliquer sur "Terminer" pour voir le résultat. Si vous avez trouvé tous les atomes une animation se lancera pour vous féliciter, si au contraire vous n'avez trouvé aucun atome une animation montrant que vous avez perdu se lancera.


PARTIE MULTI

Ce mode de jeu vous permet de jouer contre un autre joueur. Lorsque vous cliquez sur partie multi vous avez le choix entre héberger une partie ou rejoindre une partie.
Si vous hébergez la partie il vous faudra entrer un numéros de port sur lequel un autre joueur pourra se connecter pour rejoindre la partie. Ensuite vous pourrez créer un plateau de jeu et y placer les atomes que l'autre joueur devra trouver. Vous pouvez sauvegarder le plateau que vous avez créé ou charger un plateau déjà sauvegardé, les sauvegardes sont enregistrés dans le dossier "Sauvegardes". Une fois terminé vous pourrez cliquer sur "Terminer" et attendre qu'un joueur rejoigne la partie. Durant la partie vous pourrez voir tous ce que fait l'autre joueur, le pointeur de sa souris, les cases sur lesquels il clique, le parcours des laser, ainsi que la position de chaque atomes. Vous ne pourrez pas interragir dans la partie. Le chat vous permettra d'échanger avec l'autre joueur.
Si vous rejoignez une partie vous devrez trouver les atomes positionnés par le joueur qui héberge la partie. Vous ne pourrez pas voir les lasers, uniquement le début et la fin du parcours du laser. Vous pourrez changer de pseudonyme, et lancer le chat. Lorsque vous aurez terminé vous pourrez voir quels atoimes vous n'avez pas trouvé.

SCORES

En cliquant sur "Scores" vous pourrez voir l'historique des scores.

OPTIONS

Si vous accédez aux options depuis le menu, vous pourrez soit changer de fond, soit changer la musique ou en modifier le volume.
Si vous accédez aux options depuis une partie vous pourrez en plus changer de niveau, ou retourner au menu.
